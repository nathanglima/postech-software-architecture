##### Clonar o projeto
```sh
git clone https://github.com/nathanglima/postech-software-architecture.git
```
##### Instalar as dependências
```sh
npm install
```
##### Copiar o env de exemplo
```sh
cp .env_example .env
```
##### Iniciar serviços containerizados com docker
```sh
docker compose up -d
```
